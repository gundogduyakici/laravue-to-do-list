## Laravue To Do List

A simple To Do List application that works on a database prepared with Laravel and Vue.js.

- Add record.
- Check record.
- Delete record.

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## Setup

- Download and install the project.
- Then run the `npm run hot` command.
